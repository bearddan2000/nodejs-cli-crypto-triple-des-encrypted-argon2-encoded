# nodejs-cli-crypto-triple-des-encrypted-argon2-encoded

## Description
A bases for encrypting with triple des
and encoding a password with argon2.
By encrypting we guarantee a strong
password.

## Tech stack
- nodejs

## Docker stack
- node:latest

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
